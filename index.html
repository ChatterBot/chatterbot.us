Welcome to the ChatterBot homepage, here you can find articles and other information related to the ChatterBot project.

The library documentation is available at https://docs.chatterbot.us.

Additionally, you can find the code repository for ChatterBot on GitHub at https://github.com/gunthercox/chatterbot.

Related Projects

Salvius is an open source humanoid robot built using ChatterBot. For more details about the project visit salvius.org.

mathparse is a Python library for solving mathematical equations contained in strings, it is used by ChatterBot to parse mathematical expressions, but can also be used independently. For more details about the project visit mathparse.chatterbot.us.
